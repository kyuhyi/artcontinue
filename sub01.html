본 템플릿은 서브 메뉴를 하나로 관리할 수 있도록 제작되었습니다. 서버 등록 시 include를 이용하여 관리할 수 있도록 제작되었습니다.

PNG 픽셀 기본형과 흰색 아이콘이 44개 포함되어 있습니다.

메인 및 서브 공통영역 - 상단 메뉴는 서버 등록 시 중복으로 사용되는 영역입니다.

index.html 및 각 서브 페이지에 적용해 두었습니다. 서버 등록 시 include로 처리합니다. <? include "$DOCUMENT_ROOT/inc/gnb.htm"; ?>

메인 및 서브 공통영역 - 하단 footer 영역은 서버 등록 시 중복으로 사용되는 영역입니다.

index.html 및 각 서브 페이지에 적용해 두었습니다. 서버 등록 시 include로 처리합니다. <? include "$DOCUMENT_ROOT/inc/footer.htm"; ?>

서브 공통영역 - 서브페이지 lnb 영역은 서버 등록 시 중복으로 사용되는 영역입니다.

각 서브 페이지에 적용해 두었습니다. 서버 등록 시 include로 처리합니다. <? include "$DOCUMENT_ROOT/inc/lnb.htm"; ?>

서브 공통영역 - 서브페이지 오른쪽 퀵메뉴 영역은 서버 등록 시 중복으로 사용되는 영역입니다.

각 서브 페이지에 적용해 두었습니다. 서버 등록 시 include로 처리합니다. <? include "$DOCUMENT_ROOT/inc/quick_menu.htm"; ?>

서브 공통영역 - 서버에 등록하여 include 처리 시 서브 제목 및 각 lnb 영역은 스크립트로 호출합니다.

include 처리 시 각 서브 페이지 제목 및 서브 메뉴는 스크립트로 처리되었습니다.
COMPANY 메뉴 - 스크립트 부분에 $("#sub01").css("display","block");
PRODUCT 메뉴 - 스크립트 부분에 $("#sub02").css("display","block");
CUSTOMER 메뉴 - 스크립트 부분에 $("#sub03").css("display","block");
PR CENTER 메뉴 - 스크립트 부분에 $("#sub04").css("display","block");
COMMUNITY 메뉴 - 스크립트 부분에 $("#sub05").css("display","block");
SHOPPING 메뉴 - 스크립트 부분에 $("#sub06").css("display","block");

템플릿에 사용된 스크립트

모든 스크립트는 common.js 에서 확인할 수 있습니다.
